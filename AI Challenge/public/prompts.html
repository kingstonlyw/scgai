How to use these prompts (example)

Reference only

This is a visual example showing how an employee might take a prompt from this library and combine it with a starter input in Python. This code is for reference only and does not run on this page.

Example Python usage
    # Example usage of a prompt from this library in Python

    from openai import OpenAI
    import os
    import json

    # Load your API key from a .env file or environment variable
    # Example .env entry:
    # OPENAI_API_KEY=your_api_key_here
    OPENAI_API_KEY = os.getenv("OPENAI_API_KEY")

    client = OpenAI(api_key=OPENAI_API_KEY)

    # Starter input copied from this page and filled in by the user
    STARTER_INPUT = {
        "role": "Analyst",
        "context": "Capital markets inbox triage",
        "rules_or_requirements": "Route repo-related emails to Capital Markets",
        "content": "Paste the email or document text here",
        "output_preferences": "Concise bullets, include draft reply"
    }

    PROMPT_TEMPLATE = """
    You are my Inbox Triage Assistant.
    Given the email/thread content and my routing rules, produce a structured triage result:
    classification, urgency, recommended owner/team, dedupe signals, required actions with deadlines,
    and (if requested) a short draft reply in my tone.
    """

    response = client.responses.create(
        model="gpt-5",
        instructions=PROMPT_TEMPLATE,
        input=json.dumps(STARTER_INPUT)
    )

    print(response.output_text)

Tip: Employees typically copy the prompt template and starter input, paste them into their own script or notebook, and adjust the fields as needed.
